1. Running the code using Terminal:

a. Ensure that you have the following libraries installed, with at-least the following versions:

Jython 2.7.1
Python 2

b. cd into ABAGAIL/jython


c. In the Terminal, type  ./run.sh

This should run the neural network wights optimization problem and the three other optimization problems. Output will be displayed on the Terminal.
