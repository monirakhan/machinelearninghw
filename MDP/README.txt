
In order to download the requirements for the cliff grid world code run:
pip install -r requirements.txt

In order to run the tit-for-tat code have the following requirements:
matplotlib==2.0.0
numpy==1.14.0
pymdptoolbox==4.0b3 (see https://github.com/sawcordwell/pymdptoolbox)
scipy==0.18.1
Python 3.5.2

In order to run the Cliff Grid World experiments and plot run the commands:
python3 run_experiment.py --all
python3 run_experiment.py --plot

In order to run the Tit-for-Tat code run the command: 
python3 tit_for_tat.py
